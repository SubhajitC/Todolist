Explanation of the folder structure and key files:

src/config/: Configuration files for setting up the MongoDB database connection.

database.js: Contains the MongoDB database connection setup.
index.js: Imports and exports the database configuration.
src/controllers/: Controllers for handling the application's logic and interacting with the database.

taskController.js: Contains logic for tasks-related operations (CRUD).
userController.js: Contains logic for user-related operations (e.g., user registration and login).
src/models/: MongoDB models for defining the data structure and schema.

Task.js: Defines the schema for tasks.
User.js: Defines the schema for users.
src/routes/: Routes for defining API endpoints and routing requests to controllers.

taskRoutes.js: Defines routes for tasks-related API endpoints.
userRoutes.js: Defines routes for user-related API endpoints.
src/app.js: The main entry point for your Node.js application. Sets up Express.js, middleware, and routes.

.env: Configuration file for environment variables. You can store sensitive 
information like database connection details here.

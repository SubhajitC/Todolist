To download MongoDB on a Windows PC, you can do the following:
1)Go to the MongoDB Download Center.
2)Select the appropriate installer file from the dropdown menus.
Click Download.
1)Double-click the installer file.
2)Follow the prompts to install MongoDB.
3)Accept the End-User License Agreement and click the next button.
Create the directory where MongoDB will store its files.
1)Start the mongodb daemon by running C:\mongodb\bin\mongod.exe in the Command Prompt.
Test your installation.
